# Linkable, Evolving Content

A short description of what this project is and why we're doing it.

## Tasks

* [ ] Add two tables "lists" and "books"
* [ ] A list is a collection of items, e.g. customers, the attributes should be specific to the list
* [ ] a book is a collection of markdown documents; similar to a notes style app with organization and a table of contents
* [ ] both lists and books can link to, and be linked from to-dos
* [ ] the agent needs new tools to work with lists and books
* [ ] A list item, or a page in a book, can be created from a todo, for instance I should be able to create a new page in a book from my daily standup meeting where i log my notes
